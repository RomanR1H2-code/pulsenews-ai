# PulseNews AI

Это простое демонстрационное веб‑приложение, позволяющее просматривать подборку новостей, сгруппированных по категориям, с возможностью поиска и коротких аннотаций. Для управления доступом используется простейший вход в личный кабинет (логин/пароль сохраняются в локальном хранилище браузера).

## Содержимое проекта

- `frontend/index.html` – основная страница приложения. Внутри подключены React и ReactDOM через CDN и определены компоненты для аутентификации, списка новостей и фильтрации.
- `frontend/styles.css` – стили для приложения.
- `data/news.json` – пример данных с новостями. Вы можете заменить содержимое файла актуальными статьями и расширить список категорий.

## Как запустить

1. Скопируйте папку `pulsenews_ai_product` в рабочую директорию веб‑сервера или откройте файл `frontend/index.html` в браузере напрямую (поддерживаются современные браузеры). При запуске через веб‑сервер убедитесь, что путь к `data/news.json` корректен.
2. Откройте `index.html` в браузере. На экране появится форма входа. Введите любое имя пользователя и пароль (проверка условная) – и вы попадёте в личный кабинет.
3. После входа вы увидите список новостей, разбитых по категориям. Можно искать по ключевым словам и фильтровать по категориям.

## Персональные кабинеты и расширение

- **Аутентификация:** В текущей версии вход реализован упрощённо: имя пользователя сохраняется в `localStorage`. Для реального использования подключите настоящий бэкенд и систему авторизации.
- **Данные:** `news.json` содержит фиктивные новости о тенденциях в ИИ. Для интеграции с реальными источниками предусмотрите API или загрузку обновляемых данных.
- **Дизайн:** Вы можете дополнить стили или использовать сторонние библиотеки (например, Tailwind CSS) для более современного внешнего вида.

Эта версия предназначена как пример. Вы можете свободно модифицировать код, добавлять новые страницы (новостные статьи целиком, профили пользователей, админ‑панель и т. д.), интегрировать систему регистрации, push‑уведомления, аналитику и любые другие функции, необходимые для вашего проекта.
